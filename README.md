# 🤖 AutoFeat ML Pipeline

An end-to-end automated machine learning pipeline that performs:

- Data preprocessing
- Feature engineering
- Model training & comparison
- Feature importance analysis
- Prediction using user input

## 🚀 Features

- Works with ANY dataset (target = last column)
- Automatic feature engineering
- Multiple ML models (RF, LR, DT)
- Best model selection
- Streamlit UI + FastAPI backend

## 🛠️ Tech Stack

- Python
- FastAPI
- Streamlit
- Scikit-learn
- Pandas
